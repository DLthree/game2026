Overview

The Geometry Wars visual mode transforms the game into a procedurally-generated neon wireframe experience inspired by the classic Geometry Wars series. All effects are rendered in real-time using WebGL shaders and Canvas 2D, with no sprite assets required.

  • 1. Wireframe Geometry Rendering
    All entities rendered as sharp vector outlines with distinct shapes: circles (player), squares with X (enemies), triangles (projectiles)
  • 2. Full Bloom / Glow Pipeline
    WebGL shader-based bloom with multi-ring sampling for intense neon glow. Bloom intensity scales with action

Configuration Options

The Geometry Wars renderer exposes a comprehensive configuration system via GeometryWarsConfig:

Toggle between wireframe outlines and filled shapes
Enable/disable WebGL bloom shader
Toggle all particle effects (trails, impacts, explosions)
Show/hide deformable grid
Toggle parallax starfield
Enable/disable expanding ring effects
Toggle screen shake and zoom
Enable/disable entity trails

Scalar Parameters

Global multiplier for all effects (0.0 - 1.0)
Note: Configuration can be accessed via game.renderSystem.getVisualStyleSystem().getGeometryWarsRenderer().getConfig() and updated with updateConfig(updates)

How to Activate

  1. Load the game at http://localhost:8000
  2. Click the blue "Style" button in the top-right corner, or press V key to cycle through visual styles
  3. Continue cycling until you reach "Style: Geometry Wars"
  4. Enjoy the full Geometry Wars experience!

Architecture

The implementation follows the project's zero-build philosophy:

  • GeometryWarsRenderer.js - Main renderer class with particle system, grid, starfield, and effect management
  • Shaders.js - Added GEOMETRY_WARS_SHADER with multi-ring bloom sampling
  • VisualStyleSystem.js - Integrated new VisualStyle.GEOMETRY_WARS enum and rendering pipeline
  • RenderSystem.js - Modified to use wireframe rendering and draw background/foreground layers
  • Game.js - Added event triggers for particles, shockwaves, camera shake, and grid deformation
No build step required: All code is plain JavaScript ES modules. Simply refresh the browser to see changes.

Performance

The Geometry Wars renderer is optimized for real-time performance:

  • WebGL shader for bloom (GPU-accelerated)
  • Efficient particle pooling and lifetime management
  • Grid deformation uses spring physics with spatial optimization
  • Configurable intensity scalar allows performance tuning
  • Individual effect toggles for accessibility and performance
